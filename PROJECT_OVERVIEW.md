1. INTRODUCTION
This document provides a comprehensive specification for the development of a full-scale web application for Global Allianz Insurance Brokers (GAIB).
The system is designed as a modern digital insurance brokerage platform tailored to the Nigerian market, enabling seamless policy purchase, claims processing, and administrative control.
The application is not an MVP — it is a complete production-ready system.
